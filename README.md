## 목표
검은사막 생활콘텐츠 **제작**을 좀 더 쉽게 알아보자!
* 공식 제작노트는 인게임에 위치 (F2누르고 껐다 켰다..😭)
* 아이템을 제작을 위한 연관관계가 조금(?) 불친절
* 웹보다는 모바일로 확인하는게 편함

> 현재는 요리정보만 있습니다. 꾸준히 연금, 가공, 제작 등을 추가할 예정입니다.
